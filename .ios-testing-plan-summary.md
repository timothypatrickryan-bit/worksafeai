## Risk Mitigation Summary

| Risk | Mitigation |
|------|-----------|
| WebSocket latency >1s | Use ngrok professional tier, early tunnel testing |
| Memory leak in real-time | Profile with Xcode, verify listener cleanup |
| Battery drain excessive | Batch WebSocket updates, disable background sync |
| Device layout issues | Test on iPhone SE, verify responsive breakpoints |
| Offline queue conflicts | Show manual merge UI, test merge logic early |
