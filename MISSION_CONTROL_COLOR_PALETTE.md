# Mission Control Color Palette
## Gray & Blue-Gray with DARKER Backgrounds

**Approved:** March 24, 2026 @ 2:18 PM EST

---

## 🎨 Final Color System

### Backgrounds (DARKER)
- `#f3f4f6` (gray-100) - Cards, modals
- `#e5e7eb` (gray-200) - Page backgrounds  
- `#d1d5db` (gray-300) - Section backgrounds

### Buttons & Accents (Blue-Gray)
- `#4b5563` (slate-600) - Primary button color
- `#374151` (slate-700) - Button hover
- `#1f2937` (slate-800) - Button active

### Text
- `#374151` (gray-700) - Primary text (dark gray)
- `#6b7280` (gray-500) - Secondary text (mid gray)
- `#9ca3af` (gray-400) - Disabled text (light gray)

### Borders
- `#d1d5db` (gray-300) - Primary borders
- `#e5e7eb` (gray-200) - Light borders

### Semantic (Muted)
- Success: `#059669` (emerald-600)
- Warning: `#d97706` (amber-600)
- Danger: `#dc2626` (red-600)
- Info: `#4b5563` (slate-600)
