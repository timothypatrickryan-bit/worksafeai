## Security Notes

- App password is generated by Google (not the actual Gmail password)
- Credentials are stored in `.email-config.json` (git-ignored)
- Never commit raw credentials to git
- Rotate app password periodically for security
- Use environment variables instead of hardcoding

## Troubleshooting

If emails aren't sending:
1. Check that 2-Factor Authentication is enabled on the Gmail account
2. Verify the app password is correct
3. Ensure "Less secure app access" is disabled (use app passwords instead)
4. Check backend logs for SMTP connection errors
