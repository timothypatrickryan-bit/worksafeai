================================================================================
API ENDPOINTS
================================================================================

PUBLIC (no auth required):
  GET  /api/health                    - Health check

AUTHENTICATION:
  POST /api/auth/register             - Register new user
  POST /api/auth/login                - Login & get JWT token
  POST /api/auth/refresh              - Refresh expired token

PROTECTED (require JWT):
  GET  /api/status                    - Dashboard statistics
  GET  /api/gap-analysis/scores       - System health scores
  GET  /api/tasks                     - List all tasks
  POST /api/tasks                     - Create new task
  GET  /api/tasks/:id                 - Get single task
  PATCH /api/tasks/:id                - Update task

RESPONSE FORMAT:
  All responses: { success: boolean, data?: any, error?: string, timestamp: string }

================================================================================
SUPPORT & TROUBLESHOOTING
================================================================================

COMMON ISSUES:

"Port 3000 already in use"
  → Kill: lsof -i :3000; kill -9 <PID>

"Database locked error"
  → Reset: rm -rf data/; npm run seed

"Module not found errors"
  → Reinstall: rm -rf node_modules package-lock.json; npm install

"CORS errors from iOS app"
  → Check: CORS_ORIGIN in .env matches your domain

"Type errors"
  → Run: npm run type-check

SEE DOCUMENTATION:
  → README.md - Troubleshooting section
