## Key Decisions Made

### 1. **Delete Half-Built Features**
- **Inbox:** Non-functional message queue → Removed entirely
- **Alerts:** Static alert display with no real data → Removed entirely
- **Principle:** Don't ship incomplete features; only build what's genuinely useful

### 2. **iOS App: Local Development, Not App Store**
- Saves 2 weeks of build/sign/TestFlight process
- Focuses on what you actually need (app on your phone)
- Tunnel configuration handles the complexity
- Timeline: 3 weeks vs 5 weeks

### 3. **Backup System: Invest Early**
- Protected against repeated code/data corruption issues
- 5-minute recovery vs hours of debugging
- Cost: 10 minutes setup, pays for itself immediately

### 4. **Orchestration Principle: Full Execution, Not Fragments**
- Don't start work without completing it
- All team pieces (design + architecture + infrastructure) deployed together
- No waiting between sequential steps; run parallel
- Demo'd 3-person team executing 5 parallel tasks in 10 seconds
