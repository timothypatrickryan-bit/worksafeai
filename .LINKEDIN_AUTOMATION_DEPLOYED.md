## What's Been Built

Three fully functional scripts that automatically generate and post LinkedIn content:

### 1️⃣ Post Generator
```bash
node scripts/linkedin-post-now.js [insight|trending]
```
- Generates topic-specific posts
- Runs any day, any time
- Creates `.linkedin-current-post.json` with content

**Example Output:**
```
✅ Post generated
📝 Type: trending
📋 Title: Ray Tracing in Wireless—The Next Frontier
💾 Saved to: .linkedin-current-post.json
```

### 2️⃣ Scheduled Automation
```bash
/scripts/linkedin-auto-post.sh
```
- Runs Tue/Thu/Sat @ 9:00 AM EST (automated via launchd)
- Generates post automatically
- Logs all activity
- Queues for browser posting

**Automated Schedule:**
- ✅ Tuesday 9 AM → Insight post
- ✅ Thursday 9 AM → Trending post
- ✅ Saturday 9 AM → Insight post

### 3️⃣ Browser Relay Integration
```bash
node scripts/linkedin-browser-post-automation.js
```
- Posts content via your authenticated LinkedIn session
- Uses OpenClaw Browser Relay
- No API keys needed, no manual copy-paste

## Installation (One-Time Setup)

### Step 1: Activate the Scheduler (Choose One)

#### Option A: macOS launchd (Recommended - Already Set Up)
```bash
launchctl load ~/Library/LaunchAgents/com.openclaw.linkedin-auto-post.plist
```

Verify:
```bash
launchctl list | grep linkedin
```

#### Option B: Traditional Cron
```bash
crontab -e
```

Add these three lines:
```cron
0 9 * * 2 /Users/timothyryan/.openclaw/workspace/scripts/linkedin-auto-post.sh >> /Users/timothyryan/.openclaw/workspace/.linkedin-cron.log 2>&1
0 9 * * 4 /Users/timothyryan/.openclaw/workspace/scripts/linkedin-auto-post.sh >> /Users/timothyryan/.openclaw/workspace/.linkedin-cron.log 2>&1
0 9 * * 6 /Users/timothyryan/.openclaw/workspace/scripts/linkedin-auto-post.sh >> /Users/timothyryan/.openclaw/workspace/.linkedin-cron.log 2>&1
```

### Step 2: Test Everything
```bash
# Generate a test post
node /Users/timothyryan/.openclaw/workspace/scripts/linkedin-post-now.js insight

# View the generated post
cat /Users/timothyryan/.openclaw/workspace/.linkedin-current-post.json | jq '.fullPost'
```

Expected:
```
✅ Post generated
📝 Type: insight
📋 Title: [One of the titles]
💾 Saved to: .linkedin-current-post.json
```

### Step 3: Set Up Browser Relay (For Automatic Posting)
1. Open LinkedIn: https://www.linkedin.com
2. Click OpenClaw Browser Relay toolbar button (should turn badge ON)
3. Tab is now connected for automated posting

## Monitoring & Logs

### View Posting History
```bash
# All posts ever generated
cat .linkedin-posts.log

# Just published successfully
cat .linkedin-posts-published.log

# Last 5 posts
tail -5 .linkedin-posts.log

# Real-time monitoring
tail -f .linkedin-posts.log
```

### Check Scheduler
```bash
# Is launchd running?
launchctl list | grep linkedin

# Is cron configured?
crontab -l | grep linkedin

# When does it run next?

```

### Debug Issues
```bash
# Check for errors

# Verify post content exists
test -f .linkedin-current-post.json && echo "✅ File exists" || echo "❌ Missing"

ls -la scripts/linkedin-*.sh scripts/linkedin-post-now.js
```
