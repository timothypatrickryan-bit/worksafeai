USAGE EXAMPLES
================================================================================

Delegate Task via UI:
  1. Go to http://localhost:3001/team
  2. Click "📋 Delegate Task" on any agent card
  3. Fill in form:
     - Task Name: "Build API endpoint"
     - Description: "Create REST endpoints for user authentication"
     - Priority: "High"
     - Estimated Hours: "8"
     - Due Date: "2026-04-05"
     - Assign To: "jarvis" (selected)
  4. Click "Create Task"
  5. See confirmation
  6. Task assigned!

Delegate Task via API:
  curl -X POST http://localhost:3001/api/tasks \
    -H "Content-Type: application/json" \
    -d '{
      "name": "Build API endpoint",
      "description": "Create REST endpoints",
      "priority": "High",
      "estimatedHours": 8,
      "dueDate": "2026-04-05",
      "assignedTo": "jarvis"
    }'

Check Agent Tasks:
  curl http://localhost:3001/api/agents/jarvis/tasks

Get All Tasks:
  curl http://localhost:3001/api/tasks

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Modal won't open
Solution:
  - Refresh page (F5)
  - Check browser console (F12)
  - Server running? curl http://localhost:3001/api/agents

Issue: Task not creating
Solution:
  - Check Task Name is filled (required)
  - Check Agent is selected (required)
  - Check browser console for errors
  - Verify server logs

Issue: Server not responding
Solution:
  - pkill -f "node server"
  - npm start
  - Wait 3 seconds for startup

Issue: Build failing
Solution:
  - npm install
  - rm -rf node_modules
  - npm install && npm run build
