## Common Gotchas

1. **WebSocket latency high?**
   - Check ngrok tunnel status
   - May need ngrok Pro tier
   - Try Cloudflare as alternative

2. **Memory leak in real-time updates?**
   - Verify WebSocket listeners removed on unmount
   - Check Redux cleanup
   - Profile with Xcode Memory Debugger

3. **Battery drain too high?**
   - Reduce WebSocket update frequency
   - Disable background sync
   - Check for tight polling loops

4. **Tests fail on device but pass in simulator?**
   - Network latency differences
   - Storage API differences
   - Check simulator vs device iOS version

5. **Responsive layout broken on iPhone SE?**
   - Check min width constraints
   - Test landscape orientation
   - Verify text doesn't overflow
