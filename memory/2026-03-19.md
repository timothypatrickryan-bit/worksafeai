## Technical Details

### Fixed API URL Pattern
**Before:** `http://localhost:3000/api/team`  
**After:** `/api/team`  
**Why:** Relative URLs work in all environments (dev, staging, production)

### Fixed useEffect Pattern
**Before:** `useEffect(() => {...}, [state])`  
**After:** `useEffect(() => {...}, [])`  
**Why:** Empty array prevents constant re-fetches

### Fixed Data Structure
**Before:** Single object `{analysis: {...}}`  
**After:** Array of objects `[{date, content}, ...]`  
**Why:** Easier to map over daily files
