## Key Learning

**Why briefings stalled:** System creates briefings with `auto-executing` status, but no process existed to detect & spawn agents for them. The autonomy loop processes tasks, not briefings. 

**Fix:** Created dedicated briefing monitor that runs every 30 min, detects queued/auto-executing briefings, and spawns appropriate agents. Now there's an explicit handler.

**Principle:** Every automation state needs an explicit handler, or it becomes a deadlock.
