API ENDPOINTS ✅
================================================================================

1. POST /api/billing/subscribe
   - Create subscription for company
   - 3-day trial automatically activated
   - Returns: subscription_id, status, trial_end

2. GET /api/billing/status
   - Get current billing status & tier limits
   - Returns: current_tier, status, billing_period_end, tier_limits

3. POST /api/billing/change-tier
   - Change subscription tier (upgrade/downgrade)
   - Returns: subscription_id, new_tier, status

4. POST /api/billing/cancel
   - Cancel subscription at period end (soft cancel)
   - Returns: message, cancel_at_period_end flag

5. POST /api/billing/webhook
   - Stripe webhook receiver
   - Handles: subscription_created, subscription_updated, subscription_deleted,
             invoice.payment_succeeded, invoice.payment_failed
   - Signature verified, raw body required
