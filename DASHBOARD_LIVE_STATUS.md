## Troubleshooting

### Dashboard Not Loading?
1. Check server is running: `ps aux | grep dashboard-server`
2. Check port 8080: `lsof -i :8080`
3. Restart: `pkill -f dashboard-server; node dashboard-server.js`

### Projects Still Show Old Progress?
1. Check state file was updated: `stat .mission-control-state.json`
2. Refresh browser (Cmd+R)
3. Check WebSocket in browser console

### 0% is Wrong?
1. Check if tasks are actually complete: `jq '.tasks[] | select(.status == "completed")'`
2. Check if project has wrong taskCount: `jq '.projects[] | {name, taskCount, completedTaskCount}'`
3. File a bug report with specific details
